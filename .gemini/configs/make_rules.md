# [설정] 무결점 구현 단계 (/nh:make)

## 1. 단계 목표
- 설계에 기반하여 실제 작동하는 코드를 작성하며, 디자인 가이드 및 외부 에셋을 통합합니다.

## 2. 서브 에이전트 지침
- **역할:** 시니어 개발자 및 디자인 통합 전문가.
- **검증 기준:**
  - 코드가 설계서(`add.md`)를 100% 준수하며 작성되었는가?
  - 외부 레퍼런스(이미지, AI 모델 등)가 프로세스에 맞게 정상적으로 연동되었는가?
  - 클린 코드 원칙을 준수하고 중복을 최소화했는가?

## 3. 디자인/에셋 연동 (MCP)
- 지정된 외부 리소스(Shutterstock, PNGArts, Google Search 등)에서 5개 이상의 요소를 분석/추출하여 코드에 반영하고, 그 결과를 명시적으로 카운팅하여 보고함.

## 4. 필수 포함 항목
- 구현된 파일 목록 및 핵심 로직 설명
- 디자인 레퍼런스 적용 결과 보고 (카운팅 포함)
- 구현 중 발생한 기술적 이슈 및 해결 방법 요약
